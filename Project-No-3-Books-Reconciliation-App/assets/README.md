# Assets Folder

Drop your submission media here before turning in the project:

- `dashboard.png` - screenshot of the Overview page (top metrics + charts)
- `reconciliation_table.png` - screenshot of the Reconciliation page's
  transactions table (with status highlighting visible)
- `followup_list.png` - screenshot of the Reports page's follow-up list
- (optional) `demo_clip.mp4` or `.gif` - a short screen recording clicking
  through Overview → Reconciliation → Reports

Once added, embed them in `README.md`'s Screenshots section using:

```markdown
![Dashboard](assets/dashboard.png)
```
